# To-do-list-using-console
To-Do List Application
Overview
This project is a simple To-Do List application built using HTML and JavaScript. It demonstrates fundamental concepts such as arrays, conditional statements (if-else), and user input handling using the prompt function in the console.

Features
Add new tasks to your to-do list.
Mark tasks as completed.
Remove tasks from the list.
View your current list of tasks in the console.
Technologies Used
HTML
JavaScript (ES6)
Getting Started
Prerequisites
To run this application, you only need a modern web browser (e.g., Chrome, Firefox, Safari) that supports JavaScript.
